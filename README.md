## Setup

1. Clone this repository
2. Add your background music as `perfect.mp3` in the root folder
3. Open `index.html` in a browser or deploy to Vercel

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Click Deploy

## Customize

- Change the music: Replace `perfect.mp3` with your preferred song
- Edit messages: Modify the text in each `<div class="stage">` section
- Adjust colors: Update CSS variables in `:root`
